Introdução à Topologia de Redes

A topologia de rede é o sistema que mostra como os hosts estão conectados na rede. Ela define a estrutura física e lógica da rede, bem como a forma como os dados são transmitidos entre os hosts. A topologia física refere-se à disposição física dos dispositivos de rede, enquanto a topologia lógica refere-se à organização dos hosts e dispositivos de rede na rede.

As topologias de rede mais comuns são a barramento, estrela, anel e malha. Cada uma delas tem seus benefícios e desvantagens:

  • Barramento: Todos os hosts estão conectados diretamente a um único dispositivo central, como um switch. Aumenta a velocidade de transmissão de dados, mas limita a quantidade de hosts que podem ser conectados.
  • Estrela: Um único dispositivo central, como um switch, conecta todos os hosts. Aumenta a quantidade de hosts que podem ser conectados, mas a velocidade de transmissão de dados é limitada pela velocidade do dispositivo central.
  • Anel: Os hosts são conectados em uma linha circular, com cada host conectado a dois outros hosts. Aumenta a quantidade de hosts que podem ser conectados e a velocidade de transmissão de dados, mas a rede fica vulnerável a falhas em um único link.
  • Malha: Os hosts são conectados em uma rede de forma que cada host esteja conectado a todos os outros hosts. Aumenta a quantidade de hosts que podem ser conectados e a velocidade de transmissão de dados, mas a complexidade da malha torna difícil a manutenção e a expansão da rede.

IPv4, Máscara de Rede e Gateway

O IPv4 (Internet Protocol version 4) é o protocolo que permite que hosts trafeguem na internet. Ele utiliza endereços IP para identificar e direcionar pacotes de dados. No entanto, o número limitado de endereços IPv4 (aproximadamente 4 bilhões) está esgotando, levando ao desenvolvimento do IPv6.

A máscara de rede é um "bairro" que agrupa IPs. Ela é usada para determinar a parte da rede e a parte do host de um endereço IP. O gateway é um dispositivo que permite a comunicação entre redes. Ele é responsável por encaminhar pacotes de dados entre redes e hosts.

A configuração de clientes na rede pode ser feita de duas maneiras: dinâmica, em que o cliente solicita automaticamente um endereço IP ao gateway, e estática, em que o administrador atribui manualmente um endereço IP ao cliente.

IPv6

O IPv6 (Internet Protocol version 6) é uma nova versão do protocolo IP que foi desenvolvida para resolver o problema da escassez de endereços IPv4. Ele utiliza endereços IP de 128 bits, o que permite um número praticamente infinito de endereços IPv6. Os endereços IPv6 são divididos em três tipos: unicast, multicast e anycast.

O cabeçalho IPv6 contém informações como o tipo de pacote, o tamanho do pacote, a ID do fluxo e a vida útil do pacote
